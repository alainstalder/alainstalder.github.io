          list of services, never null

        • @java.lang.Override ServiceState getState()

          Get service state of the group.

          Returns:
          If there is no service in the group, OFF is returned, if all services are in the same state, that state is returned, else BUSY_STARTING, BUSY_STOPPING, BUSY_EVENT, IDLE, OFF is returned, in that order of priority, if it is the state of at least one service.

        • @java.lang.Override void start()

          Start all services in the group. Runtime exceptions are not caught, hence if the first service throws while starting up, no attempt is made to start the others.

        • @java.lang.Override void stop()

          Stop all services in a group. Runtime exceptions are only logged, hence it is always attempted to stop all services.
